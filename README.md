isodumper
=========

A tool for writing ISO images on a USB stick. It's a fork of usb-imagewriter. 
We added also a formatting feature and a backup of USB device content.


Requirements
-------------

- coreutils
- pango
- polkit
- procps
- pygtk2.0-libglade
- python
- python-parted
- udisks
- xterm
